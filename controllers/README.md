# Documentación Frontend

## **Taller de Carpintería**&#x20;

### **Resumen del Proyecto**

Este sistema centraliza las tareas diarias de un taller en una interfaz moderna y eficiente. Permite administrar solicitudes de servicio, controlar el inventario de herramientas y gestionar al personal técnico en un solo lugar.

**El proyecto ha pasado por dos etapas clave:**

1. **Validación de Interfaz:** Prototipado funcional con persistencia local para asegurar la mejor experiencia de usuario.
2. **Integración Total:** Conexión a un Backend RESTful para la gestión de datos en tiempo real.

### Funcionalidades Clave

* Visualización en tiempo real de indicadores clave, métricas de servicios y estado general del taller.
* Control de Servicios, seguimiento de estados, registro de evidencia fotográfica, informacion de solicitantes, etc.
* Gestión de herramientas con programación de mantenimientos.
* Administración de técnicos y asignación de tareas.
* Rutas protegidas y autenticación por tokens (JWT).
* Administración de perfiles, técnicos asignables y control de acceso al sistema.

### Stack Tecnológico

* **Core:** React (v18) + Vite + TypeScript.
* **Estilos:** Tailwind CSS + Radix UI.
* **Comunicación:** Axios para consumo de API REST.
