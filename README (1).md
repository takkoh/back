# Documentación Backend

## CarpinteriaSF

### Descripción:

API REST para gestion del taller de carpinteria para la Universidad de Sonora.

### Dependencias

* bcryptjs versión 3.0.3
* cors 2.8.6
* dotenv 16.6.1
* express 4.22.1
* express-session 1.19.0
* mysql2 3.22.3
* node-cron 4.21

### Dependencias DEV

* nodemon 3.1.14
